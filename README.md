FizzBuzzJS
==========
Simple, testable, extendable FizzBuzz using JavaScript.

Setup
-----
1. Install Git, Node.js, NPM.
2. Clone this repository: `git clone [url]`.
3. Run `npm install` in the root.

Running
-------
* Node
	* Use `node main.js`.
* Browser
	* Visit `index.html` using a browser.

Testing
-------
* Once
	* Use `npm test`.
* Continuous
	* Use `make test-w`.
